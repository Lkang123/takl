# App icons directory

Place your app icons here. Recommended files:

- whisp-192.png (PNG, 192x192)
- whisp-512.png (PNG, 512x512)
- Optional: whisp-maskable-512.png (PNG, 512x512 with safe padding for maskable icons)

After you add the files, I can update public/manifest.webmanifest to reference them and (optionally) add <link rel="icon"> / <link rel="apple-touch-icon"> in public/index.html.
